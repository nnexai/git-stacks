# @git-stacks/web

The browser-only git-stacks client. It builds to a self-contained `file:` asset, consumes a one-use fragment, creates an ephemeral in-memory P-256 key, pins browser-native WebTransport to the loopback helper, and renders the path- and secret-minimized projection plus service-owned terminals. It uses no HTTP API, cookie principal, worker, or durable browser product storage.
