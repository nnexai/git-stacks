<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI flags -> runner scheduler | Local user input controls concurrency and mode selection. |
| Child test processes -> buffered report | Unstructured stdout/stderr from child processes must be attributed without interleaving. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-93-1-01 | Denial of service | `scripts/test-runner-core.ts` worker parsing | mitigate | Reject invalid worker counts and default to 4 per D-01 and D-02 rather than using CPU-count auto-scaling. |
| T-93-1-02 | Repudiation | Integration output reporting | mitigate | Buffer per-file stdout/stderr and print completion-order blocks plus a final failed-file summary per D-05, D-06, and D-07. |
| T-93-1-03 | Availability | Integration suite execution | mitigate | Drain the full queue and report all failures instead of failing fast per D-13. |
</threat_model>
