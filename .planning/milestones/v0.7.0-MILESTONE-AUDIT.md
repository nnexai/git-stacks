### Integration Notes

**POLISH-07 folder deletion design:** CLI `clean` command intentionally uses inline folder deletion with a user prompt (`workspace.ts:396-416`) rather than the library's `deleteFolder` opt. This is by design — `clean` needs interactive confirmation between worktree removal and folder deletion, which the library function can't provide. The `deleteFolder` opt serves programmatic callers (`removeWorkspace`, `mergeWorkspace`) that always delete. Not a gap.
