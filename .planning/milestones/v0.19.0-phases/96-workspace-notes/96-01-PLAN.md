| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-96-01 | Tampering | `src/lib/notes.ts` parse-and-append path | mitigate | Parse the full existing JSONL file before append or clear per D-12 so malformed stores block mutation instead of being silently extended. |
| T-96-02 | Tampering | `src/lib/notes.ts` note path helper | mitigate | Keep note files rooted under `NOTES_DIR` and reject workspace path traversal so storage cannot escape the config root. |
| T-96-03 | Information Disclosure | `src/lib/notes.ts` summary helper | accept | Notes are intentionally local operator metadata under the user's own config root; Phase 96 adds no cross-user sharing or remote transport. |
