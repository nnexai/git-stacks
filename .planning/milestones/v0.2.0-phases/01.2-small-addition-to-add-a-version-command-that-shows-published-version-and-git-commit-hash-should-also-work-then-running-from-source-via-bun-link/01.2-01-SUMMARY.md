## Accomplishments
- Created `src/lib/version.ts` with `getVersionString()` that reads version from package.json at runtime
- Appends 7-char git commit hash in parentheses when inside a git repo, with `-dirty` suffix when working tree has uncommitted changes
- Returns plain semver when outside a git repo (e.g., npm global install)
- Replaced hardcoded `"0.1.1"` in `src/index.ts` with dynamic version call — now in sync with package.json `"0.1.3"`
- 4 tests cover all scenarios: git repo (clean), git repo (dirty), no git repo, package.json version match
