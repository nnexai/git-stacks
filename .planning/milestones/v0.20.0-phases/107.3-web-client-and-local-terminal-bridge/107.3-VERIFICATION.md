# Phase 107.3 Verification

## Verified Behavior

- Pairing is single-use, same-origin, loopback-only, and credential-isolated.
- Workspace DTOs omit sensitive launch material.
- Multiple PTYs accept input, resize correctly, retain independent output, and restore focus on tab selection.
- Hidden terminals pause bulk output transmission and replay it on resume without losing lifecycle controls or agent signals.
- Browser reload and closure reconnect to retained service sessions.
- Shell exit removes interactive tabs; configured command results remain inspectable.
- Explicit close terminates process groups and clears exact-surface activity.
- Provider icons are deduplicated in the sidebar while tab-level state remains exact.
- Terminal, replay, socket, event, operation, and principal resources are bounded.

## Automated Evidence

- Focused web security, terminal, signal, projection, principal, presentation, and provider tests.
- Full repository test suite.
- Root and browser TypeScript checks.
- Web production build.
- Dependency-cycle and release gates.

## Honest Limits

- Browser terminals are process-local to the managed service.
- Remote access and shared writers are not supported.
- Platforms without actual-host PTY and process-tree verification remain capability-gated.
