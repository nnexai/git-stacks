# Plan 55-02: Install Command --copilot and --claude Flags

<objective>
Add `--copilot` and `--claude` flags to `git-stacks install --hooks` that bypass the interactive multi-select prompt and directly install the specified hook set(s). When neither flag is given, preserve the existing interactive chooser. When both flags are given, install both simultaneously.
</objective>

4. **Replace the multi-select block** with flag-based resolution. Create a helper function `resolvePlugins` above the command definition:

   ```typescript
   function resolvePlugins(
     opts: { copilot?: boolean; claude?: boolean },
     allPlugins: AgentHookPlugin[]
   ): AgentHookPlugin[] {
     const hasCopilot = opts.copilot === true
     const hasClaude = opts.claude === true

     if (!hasCopilot && !hasClaude) {
       // No flags â€” return empty to signal "use interactive prompt"
       return []
     }

     const selected: AgentHookPlugin[] = []
     if (hasClaude) {
       const claude = allPlugins.find(p => p.id === "claude-code")
       if (claude) selected.push(claude)
     }
     if (hasCopilot) {
       const copilot = allPlugins.find(p => p.id === "copilot")
       if (copilot) selected.push(copilot)
     }
     return selected
   }
   ```

5. **Update the install flow** in the else branch (the non-remove path). Replace the current multi-select block:

   ```typescript
   // Resolve plugins from flags or interactive prompt
   let selectedPlugins: AgentHookPlugin[]
   const fromFlags = resolvePlugins(opts, agentHookPlugins)

   if (fromFlags.length > 0) {
     // Flags provided â€” skip interactive prompt (HOOK-01, HOOK-02, HOOK-04)
     selectedPlugins = fromFlags
   } else {
     // No flags â€” interactive multi-select (HOOK-03)
     const pluginChoices = await p.multiselect({
       message: "Select agent frameworks to install hooks for",
       options: agentHookPlugins.map((plugin) => ({
         value: plugin.id,
         label: plugin.label,
       })),
       required: true,
     })

     if (p.isCancel(pluginChoices)) {
       p.cancel("Cancelled.")
       process.exit(0)
     }

     const selectedIds = pluginChoices as string[]
     selectedPlugins = agentHookPlugins.filter((p) => selectedIds.includes(p.id))
   }

   for (const plugin of selectedPlugins) {
     plugin.install(targetDir, workspace.name)
   }

   p.log.success(
     `Installed ${selectedPlugins.map((p) => p.label).join(", ")} hooks into ${targetDir} (workspace: ${workspace.name})`
   )
   ```

6. **Import type** â€” add `AgentHookPlugin` to the imports from `../lib/agent-hooks`:

   ```typescript
   import type { AgentHookPlugin } from "../lib/agent-hooks"
   ```

The key behavioral changes:

- `git-stacks install --hooks --remove --copilot` â†’ removes only Copilot hooks
- `git-stacks install --hooks --remove` â†’ removes all hooks (unchanged behavior)
