<threat_model>
## STRIDE threat register

| Category | Threat | Disposition and mitigation |
|---|---|---|
| Spoofing | Launch overlay replaces reserved workspace/service identity. | Apply reserved `GS_*` values last through the shared composer. |
| Tampering | Snapshot-generated shell source changes quoted commands. | Delete generators and pass each command as one positional argument. |
| Repudiation | PTY/non-PTY startup failures lose stage attribution. | Preserve structured shell/mode/stage/recovery metadata. |
| Information Disclosure | Launch-time secrets or refreshed socket leak into state/output. | Resolve only at launch and assert absence from diagnostics, snapshots, and projections. |
| Denial of Service | Failed non-PTY launch leaves descendants. | Adapter-owned process group and TERM/grace/KILL cleanup. |
| Elevation of Privilege | PTY bypasses validated shell or uses stale attacker-selected socket. | Shared shell validation and atomic per-launch environment snapshot. |
</threat_model>
