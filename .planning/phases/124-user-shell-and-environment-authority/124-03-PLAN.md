<threat_model>
## STRIDE threat register

| Category | Threat | Disposition and mitigation |
|---|---|---|
| Spoofing | Browser/remote client claims local refresh authority. | Authorize authenticated transport/origin before payload parsing. |
| Tampering | Partial or oversized refresh corrupts launch state. | Strict bounded schema and atomic whole-snapshot replace. |
| Repudiation | Refresh failure has no usable diagnostic. | Return stable metadata-only result codes without values. |
| Information Disclosure | PATH/socket leak through projection, revision, log, or URL. | Keep state outside durable graphs and scan canary values across surfaces. |
| Denial of Service | Unbounded payload or rapid invalid refresh consumes service resources. | Enforce small value/key bounds before allocation and preserve prior state on failure. |
| Elevation of Privilege | Remote refresh redirects future agent/socket use. | Same-user local-only route plus allowlisted keys. |
</threat_model>
