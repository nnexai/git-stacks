<threat_model>
## STRIDE threat register

| Category | Threat | Disposition and mitigation |
|---|---|---|
| Spoofing | A workspace command substitutes a different shell. | Only the validated shared adapter selects the shell. |
| Tampering | Hook text changes the main command through concatenation. | Execute each step separately and pass each command as one argument. |
| Repudiation | Initialization and command exits are conflated. | Preserve structured stage and exact step exit data. |
| Information Disclosure | CLI diagnostics print composed environment values. | Render metadata and recovery only; retain redaction tests. |
| Denial of Service | Parallel descendants survive cancellation. | Delegate cancellation to adapter-owned process groups. |
| Elevation of Privilege | Legacy `sh` branch bypasses validation. | Delete all fallback branches and assert they are never invoked. |
</threat_model>
