    - formatError("Workspace 'foo' not found", "run: ws list") returns "error: Workspace 'foo' not found\n  \u2192 run: ws list"
    - formatError with empty string hint returns the message without hint line (same as no hint)

  <action>
Create `src/lib/errors.ts` with:

```typescript
export function formatError(message: string, hint?: string): string {
  if (hint) {
    return `error: ${message}\n  \u2192 ${hint}`
  }
  return `error: ${message}`
}
```

Create `tests/lib/errors.test.ts` with tests covering:
1. Message-only format: `expect(formatError("Workspace 'foo' not found")).toBe("error: Workspace 'foo' not found")`
2. Message + hint format: `expect(formatError("Workspace 'foo' not found", "run: ws list")).toBe("error: Workspace 'foo' not found\n  \u2192 run: ws list")`
3. Undefined hint: same result as message-only
4. Empty string hint: same result as message-only (guard against empty hint producing a bare arrow line)

Use imports: `import { describe, test, expect } from "bun:test"` and `import { formatError } from "../../src/lib/errors"`

- `console.error(\`pre_remove hook failed for '${ws.name}': ${err}\`)` -> `console.error(formatError(\`pre_remove hook failed for '${ws.name}': ${err}\`))`

**clean command (named path):**
- `console.error("Usage: ws clean <name> [--gone]")` -> `console.error(formatError("Missing workspace name", "usage: ws clean <name> [--gone]"))`
- `console.error(result.error)` -> `console.error(formatError(result.error))` (clean result error)

**remove command:**
- `console.error(\`Workspace '${name}' not found. Run \\\`ws list\\\`...\`)` -> `console.error(formatError(\`Workspace '${name}' not found\`, "run: ws list"))`

**merge command:**
- `console.error(\`Workspace '${name}' not found...\`)` -> `console.error(formatError(\`Workspace '${name}' not found\`, "run: ws list"))`

**run command:**
