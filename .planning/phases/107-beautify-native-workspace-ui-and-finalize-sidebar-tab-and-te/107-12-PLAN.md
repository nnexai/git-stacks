## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Connection/model state to user actions | Incorrect readiness or copy can invite invalid/destructive operations. |
| Visual state to assistive technology | CSS-only selection/status can conceal critical state from nonvisual users. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-107-12-01 | Elevation of Privilege | stale/incompatible actions | high | mitigate | Central GAction readiness disables mutation and exposes only retry/details while retaining read-only content. |
| T-107-12-02 | Information Disclosure | technical errors/details | medium | mitigate | Friendly primary copy, bounded details, and technical code only in logs/details. |
| T-107-12-03 | Denial of Service | narrow/large-text layout | medium | mitigate | Active 720px overlay breakpoint, clamped dialogs, and allocation tests at 200% text. |
| T-107-12-04 | Spoofing | CSS-only state | medium | mitigate | Real accessible selected/checked/status properties plus redundant text/icon/shape cues. |
</threat_model>
