<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Local filesystem/processes to service journal | Untrusted timing and mutation bursts become authenticated client-visible control events. |
| Journal to live SSE subscribers | Durable ordered records cross into bounded per-client queues and replay cursors. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-107-05-01 | Tampering | snapshot invalidation revision | high | mitigate | Derive revision only from authoritative rebuild and validate through strict event schema. |
| T-107-05-02 | Repudiation | publish-before-durable race | high | mitigate | Await serialized fsynced journal append and publish only the returned event. |
| T-107-05-03 | Denial of Service | event storms/replay | medium | mitigate | Revision coalescing, existing journal retention, bounded broker queues, rate limits, and deadlines. |
| T-107-05-04 | Information Disclosure | snapshot_busy/replay errors | low | accept | Return only bounded codes/counts/cursors/revision metadata, not paths or content. |
</threat_model>
