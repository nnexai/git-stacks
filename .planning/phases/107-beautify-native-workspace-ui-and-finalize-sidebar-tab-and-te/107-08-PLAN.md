<interfaces>
`service_client.Request` currently carries method/path/authorization/body/Last-Event-ID. Extend it with explicit content type and idempotency key rather than encoding headers inside transport call sites.

`Client.decodeAggregateSnapshot(body)` returns a reducer action; capacity violations should use `Action.capacity_exceeded` from Plan 107-07 and invalid payloads should remain errors.

The controller analog is `command_launcher.Launcher`: a pure stateful projection with no GTK types, bounded user-visible errors, and explicit readiness.
</interfaces>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Authenticated HTTP JSON to Zig | Service-sized arrays, strings, IDs, operation errors, and cursors enter bounded native memory. |
| User form to HTTP request | User-entered name/branch/source becomes JSON plus idempotency state. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-107-08-01 | Tampering | strict response decode | high | mitigate | Exact keys, ID/schema/length checks, owned copies, and fail-closed outcomes. |
| T-107-08-02 | Denial of Service | aggregate/cursor/operation sizes | high | mitigate | Preflight counts against named limits before copies and bound all strings/collections. |
| T-107-08-03 | Repudiation | ambiguous create retry | high | mitigate | Freeze one request and idempotency key until a definitive operation result. |
| T-107-08-04 | Spoofing | service requests | high | mitigate | Preserve protected descriptor/credential discovery, loopback validation, and exact Bearer header. |
| T-107-08-05 | Elevation of Privilege | JSON/form encoding | medium | mitigate | Use a JSON writer, never shell interpolation or caller-provided paths. |
</threat_model>
