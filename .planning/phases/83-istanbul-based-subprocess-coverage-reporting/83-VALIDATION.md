## Sampling Rate

- **After every task commit:** Run `bun test tests/commands/istanbul-smoke.test.ts -x` until the coverage runner exists, then run the narrowest changed coverage command (`bun run coverage:unit` or `bun run coverage:integ`)
- **After every plan wave:** Run `bun run coverage && bun run test && bun run typecheck`
- **Before `/gsd-verify-work`:** Full suite must be green and `.coverage/coverage-final.json` must exist
- **Max feedback latency:** 60 seconds
