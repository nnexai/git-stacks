<threat_model>
## Trust Boundaries

| Boundary | Description |
|---|---|
| Keyboard event to PTY/app | Incorrect consumption can execute an app mutation or steal shell input. |
| Web action to service | Repeated or disabled actions could submit unauthorized/duplicate mutations. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|---|---|---|---|---|---|
| T-125-07 | Elevation of privilege | xterm shortcut preprocessing | high | mitigate | Exact shared matcher plus handled-only consumption; rejected events return to xterm unchanged. |
| T-125-08 | Denial of service | repeat/action dispatch | high | mitigate | Keydown-only dispatch, per-action repeat policy, modal exclusivity, and one callback invocation. |
| T-125-09 | Tampering | attention target application | medium | mitigate | Apply only shared validated candidates against current projected state; never synthesize stale fallback targets. |
| T-125-SC | Tampering | dependency supply chain | low | accept | No dependency changes; existing xterm custom handler and shared packages are used. |
</threat_model>
