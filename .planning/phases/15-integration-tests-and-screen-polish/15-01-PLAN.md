**App.tsx — Replace helpBarText createMemo (lines 168-176) with width-tiered version (per D-01, D-02, D-03, D-04):**

Replace the existing `helpBarText` createMemo with:

```typescript
const helpBarText = createMemo(() => {
  const w = dims().width
  const t = tab()
  const msgShortcut = t === "workspaces" ? "  m Messages" : ""

  if (w < 50) return "? Help  q Quit"
  const core = `Enter Actions  Space Select  / Filter${msgShortcut}  ? Help  q Quit`
  if (w < 65) return core
  if (w < 80) return `r Refresh  ${core}`
  if (w < 100) return `1/2/3 Tabs  r Refresh  ${core}`
  return `\u2191\u2193/jk Navigate  1/2/3 Tabs  r Refresh  ${core}`
})
```

This implements the tier breakpoints from D-03:
- >= 100: Full bar with Navigate
- >= 80: Drop Navigate
- >= 65: Also drop Tabs
- >= 50: Also drop Refresh
- < 50: Only `? Help  q Quit`

D-04: `m Messages` only appears when `tab() === "workspaces"`.
