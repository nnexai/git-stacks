## Trust Boundaries

| Boundary | Description |
|---|---|
| Fallible local/network observations → qualification policy | Operational failures must not become confirmed cleanup evidence. |
| Concurrent probe results → volatile cache | Late or forced calls compete to write shared service-memory evidence. |
| Authoritative revision → evaluation dependencies | Stale requests must be rejected before any potentially expensive or misleading observation. |
| Evaluator dependencies → lifecycle/persistence | Advisory policy must remain unable to mutate workspace state. |

| T-127-04-02 | Tampering | Cache generation writes | high | mitigate | Per-key monotonic generations, force-read bypass, and newest-write-only deferred-race tests. |
