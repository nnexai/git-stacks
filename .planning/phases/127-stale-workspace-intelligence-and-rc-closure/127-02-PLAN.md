| Keyboard/pointer → action coordinator | Repeated or leaked input could trigger duplicate or confused-deputy lifecycle actions. |
| Release validation → Git/provider/package hosts | A local check must not become an outward release operation. |
| Deterministic tests → evidence claim | Fixtures cannot impersonate hosted, physical, authenticated, screenshot, interactive, or human proof. |

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
