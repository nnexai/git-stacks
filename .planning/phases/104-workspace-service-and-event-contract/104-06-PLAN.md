## Trust Boundaries

| Boundary | Description |
|---|---|
| Local network to service | Loopback requests are untrusted until bearer admission and strict validation. |
| Descriptor filesystem to official client | Endpoint/process metadata must be protected, atomic, fresh, and secret-free. |
| SSE client to service resources | Connections and consumption rate can exhaust sockets or memory. |
