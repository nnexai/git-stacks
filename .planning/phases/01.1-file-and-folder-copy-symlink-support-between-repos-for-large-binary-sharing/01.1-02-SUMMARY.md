## Decisions Made

- `workspace.files` is unavailable during `new` (YAML not yet written), so `applyFileOpsForWorkspace` receives `{} as Workspace` — only stack.files applies at the workspace-instance level during creation
- File op errors during `new` are fatal (abort with `process.exit(1)`) — the workspace doesn't exist yet, so a missing source is unrecoverable
- File op errors during `open` are non-fatal warnings via `onProgress` — the workspace already exists, idempotent re-apply is forgiving
- Merge regex `^([^=\\s#][^=]*)=(.*)$` handles standard KEY=value env lines while preserving comments and blank lines
