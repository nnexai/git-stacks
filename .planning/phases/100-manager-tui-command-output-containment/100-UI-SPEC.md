## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Inline gaps inside status rows and short prefixes |
| sm | 8px | Compact line separation in output viewer |
| md | 16px | Default internal modal padding |
| lg | 24px | Section breaks between status and output body |
| xl | 32px | Larger separation for completion footer copy |
| 2xl | 48px | Dialog breathing room in large terminal sizes |
| 3xl | 64px | Page-level separation where full-screen states need it |

Exceptions: none

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #000000 | Dashboard background, centered dialog surface |
| Secondary (30%) | #1f1f1f | Bordered modal body, grouped output sections, passive status rows |
| Accent (10%) | #00ffff | Running indicator, selected row marker, focused command label |
| Destructive | #ff4d4f | Failed command status, stderr-emphasis lines, destructive confirmation copy |

Accent reserved for: active command status, current selection marker, modal focus cues, and actionable footer hints only

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Run Command |
| Empty state heading | No command output yet |
| Empty state body | Start a command from the manager action menu to view captured output here. |
| Error state | Command failed. Review stderr lines, then close and retry from the action menu. |
| Destructive confirmation | Close viewer while running: This command is still running. Wait for completion before leaving this screen. |
