<action>
In `src/commands/workspace.ts`, within the `registerWorkspaceCommands(program)` function, add a new command block:

```typescript
program
  .command("env [workspace]")
  .description("Show environment variables for a workspace")
  .option("--format <format>", "Output format (table, shell, dotenv, json)", "table")
  .option("--repo <name>", "Include repo-specific variables")
  .action(async (workspace: string | undefined, opts: { format: string; repo?: string }) => {
    // ... implementation
  })
```

Implementation logic inside the action:

1. **Resolve workspace name:**
   - If `workspace` argument provided: validate with `workspaceExists()`, read with `readWorkspace()`
   - If no argument: call `detectWorkspaceFromCwd()`. On failure, print error via `formatError("Could not detect workspace from current directory", "run from inside a worktree or specify: git-stacks env <workspace>")` and `process.exit(1)`

2. **Validate format:**
   - Check `opts.format` is one of: `"table"`, `"shell"`, `"dotenv"`, `"json"`
   - If invalid: `formatError("Invalid format '${opts.format}'", "use: --format table|shell|dotenv|json")` and `process.exit(1)`

3. **Build env vars:**
   - `const globalConfig = readGlobalConfig()`
   - `const tasksDir = join(getTasksDir(globalConfig), ws.name)` (where `ws` is the read workspace)
   - `const env = buildBaseEnv(ws, tasksDir, "env")` — use `"env"` as the triggeredBy value

4. **Handle --repo flag:**
   - If `opts.repo` provided: find the repo in `ws.repos` by name. If not found: `formatError("Repo '${opts.repo}' not found in workspace '${ws.name}'", "available repos: " + ws.repos.map(r => r.name).join(", "))` and `process.exit(1)`. If found: `env = buildRepoEnv(env, repo)` (use the repo object with name, task_path, main_path)
   - If no `--repo` and no explicit workspace arg (CWD detection was used): call `detectRepoFromCwd(ws)` and if a repo is found, add its vars via `buildRepoEnv`

5. **Format and output:**
   - `import { formatEnv, type EnvFormat, detectRepoFromCwd } from "../lib/env"`
   - `console.log(formatEnv(env, opts.format as EnvFormat))`

Add these imports at the top of the file:
- `import { formatEnv, type EnvFormat, detectRepoFromCwd } from "../lib/env"`
- `buildBaseEnv` and `buildRepoEnv` from `"../lib/workspace-ops"` (add to existing import)
- `readGlobalConfig` from `"../lib/config"` (already imported)
- `getTasksDir` from `"../lib/paths"` (already imported via another import — check and add if missing)
- `join` from `"path"` (add if not already imported)

Place the command after the existing `paths` command block and before the `run` command block (or at a logical position among read-only inspection commands).
</action>
