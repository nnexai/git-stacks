<threat_model>
## Trust Boundaries
| Boundary | Description |
|---|---|
| GTK callbacks to opaque Ghostty surface | Late or cross-generation callbacks can use freed native state. |
| Host GL context to Ghostty renderer | Drawing without the correct current context can corrupt rendering state. |
| User configuration to native runtime | Untrusted configuration and paths affect process/render behavior. |
## STRIDE Threat Register
| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|---|---|---|---|---|---|
| T-105-UAF-01 | Tampering | surface callbacks | high | mitigate | Main-thread affinity, generation checks, reverse-order idempotent teardown. |
| T-105-GL-01 | Denial of Service | GtkGLArea lifecycle | high | mitigate | Current-context checks, zero-size guards, realized/unrealized tests. |
| T-105-CONFIG-01 | Information Disclosure | configuration diagnostics | medium | mitigate | Bounded error metadata without config contents or environment values. |
</threat_model>
