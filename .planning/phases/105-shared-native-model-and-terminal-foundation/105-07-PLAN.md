<threat_model>
## Trust Boundaries
| Boundary | Description |
|---|---|
| Terminal output → renderer/host events | Escape data, titles and styles are untrusted. |
| Clipboard/IME/native events → PTY | Sensitive and attacker-controlled input reaches a shell. |
## STRIDE Threat Register
| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|---|---|---|---|---|---|
| T-105-OUTPUT-01 | Denial of Service | frame/cache/event processing | high | mitigate | Bounds, clipping, cache eviction, retained-capacity reset and stress counters. |
| T-105-PASTE-01 | Elevation of Privilege | paste path | high | mitigate | Pinned safe-paste/bracketed mode and control-byte tests. |
| T-105-CLIP-01 | Information Disclosure | clipboard | high | mitigate | User action boundaries, no diagnostic bytes, generation-safe async reads. |
| T-105-ASYNC-01 | Tampering | IME/clipboard callbacks | high | mitigate | GTK-main ownership and generation rejection after teardown. |
</threat_model>
