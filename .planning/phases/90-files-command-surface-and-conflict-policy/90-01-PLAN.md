## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Workspace YAML paths -> filesystem reads | User-controlled sync paths determine which local trees are inspected. |
| Large local trees -> CLI output | Thousands of files can cross into user-facing output if uncapped. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-90-01 | Information Disclosure | Status output | mitigate | Default output shows counts only; path listings require `--verbose` and are capped. |
| T-90-02 | Denial of Service | Tree comparison | mitigate | Avoid manifest writes and cap verbose path collection; keep comparison bounded to configured source/target trees. |
| T-90-03 | Tampering | Status operation | mitigate | Status helpers are read-only and tests assert no manifest or metadata files are created. |
</threat_model>
