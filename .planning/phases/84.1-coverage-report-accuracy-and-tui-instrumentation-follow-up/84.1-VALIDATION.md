## Sampling Rate

- **After the spike:** Run both focused sentinel coverage commands serially and inspect `.coverage/coverage-final.json` before implementation proceeds.
- **After every task commit:** Run the narrowest relevant command from the verification map below.
- **After every plan wave:** Run `bun run coverage && bun run verify:gates`.
- **Before `$gsd-verify-work`:** `bun run verify` must pass.
- **Max feedback latency:** 30 seconds for unit/gate tests; full verify may be longer because it intentionally refreshes coverage.
