The milestone-level integration review confirms these flows are connected correctly:

1. Ahead/behind data flows from `git.ts` to `workspace-ops.ts`, then into `list`/`status` CLI output and dashboard row/detail rendering.

3. Labels flow from schema validation into CLI CRUD/new/list support and dashboard render/filter/group-by behavior.

5. Stash-on-sync flows from git stash primitives into `syncWorkspace()`, `git-stacks sync --stash`, JSON output, and dashboard sync progress.
6. Release prep accurately reflects shipped versioning and documented behavior.
