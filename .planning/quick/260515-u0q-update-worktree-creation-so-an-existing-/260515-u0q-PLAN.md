<objective>
Update workspace worktree creation so local workspace branches are reused and remote workspace branches are treated like `git checkout`: if `origin/<branch>` exists, create or reuse the local branch from that remote branch and configure upstream tracking.

Purpose: Workspace creation should preserve branch intent. A workspace branch that already exists locally or remotely must not be silently recreated from the wrong base commit, and later push/pull/sync flows should see the expected upstream.

Output: A focused implementation in `src/lib/git.ts` plus regression coverage using existing Bun test fixtures.
</objective>
