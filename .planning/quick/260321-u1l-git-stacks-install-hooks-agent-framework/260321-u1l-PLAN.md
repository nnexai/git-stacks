    Create `src/lib/agent-hooks/types.ts`:
    ```typescript
    export interface HookEntry {
      event: string          // "Stop", "PreToolUse", "UserPromptSubmit", "PostToolUse"
      matcher?: string       // e.g. "AskUserQuestion" (omit for events without matchers)
      command: string        // shell command to run
    }

    export interface AgentHookPlugin {
      id: string
      label: string
      /** Generate the hook entries for a given workspace */
      generateHookEntries(workspaceName: string): HookEntry[]
      /** Install hooks into a repo worktree directory */
      install(repoWorktreePath: string, workspaceName: string): void
      /** Remove this plugin's hooks from a repo worktree directory */
      remove(repoWorktreePath: string): void
    }
    ```

    Create `src/lib/agent-hooks/claude-code.ts`:
    - `generateHookEntries(workspaceName)` returns 4 entries:
      1. `{ event: "Stop", command: 'git-stacks message send "Claude has finished and may need your attention" --workspace {name} --from claude' }`
      2. `{ event: "PreToolUse", matcher: "AskUserQuestion", command: 'git-stacks message send "Claude is asking a question — input needed" --workspace {name} --from claude' }`
      3. `{ event: "UserPromptSubmit", command: 'git-stacks message clear --workspace {name} --from claude' }`
      4. `{ event: "PostToolUse", matcher: "AskUserQuestion", command: 'git-stacks message clear --workspace {name} --from claude' }`
    - `install(repoWorktreePath, workspaceName)`:
      - Read existing `.claude/settings.json` if it exists (create `.claude/` dir if needed)
      - Parse as JSON, default to `{}` if missing/empty
      - Filter out any existing hooks where `command` includes "git-stacks" (idempotency)
      - Append the 4 new entries to `hooks` array
      - Write back with `JSON.stringify(data, null, 2)`
    - `remove(repoWorktreePath)`:
      - If `.claude/settings.json` doesn't exist, return
      - Parse, filter `hooks` array to remove entries where `command` includes "git-stacks"
      - If hooks array is now empty, delete the `hooks` key entirely
      - Write back (or if settings object is now empty `{}`, delete the file)

    ```typescript
    import { Command } from "commander"
    import * as p from "@clack/prompts"
    import { listWorkspaces, readWorkspace, readGlobalConfig } from "../lib/config"
    import { getTasksDir } from "../lib/paths"
    import { agentHookPlugins } from "../lib/agent-hooks"
    import { join, resolve } from "path"

    export const installCommand = new Command("install")
      .description("Install agent framework hooks into workspace repos")
      .option("--hooks", "Install agent notification hooks")
      .option("--remove", "Remove installed hooks instead of installing")
      .action(async (opts: { hooks?: boolean; remove?: boolean }) => { ... })
    ```

    The action logic:
    1. If `--hooks` not passed, show help and exit (future: other install targets)
    2. **Detect workspace from cwd**: `detectWorkspaceFromCwd()` function (defined in install.ts, not workspace-ops.ts — this is install-specific logic):
       - Read global config to get `workspace_root`
       - Compute `tasksDir = getTasksDir(workspace_root)`
       - Get `cwd = process.cwd()`
       - If cwd starts with `tasksDir + "/"`, extract workspace name as the next path segment after tasksDir
       - Also check `process.env.WS_WORKSPACE`
       - Return workspace name or null
    3. If no workspace detected, show `p.select()` with all workspace names (from `listWorkspaces()`)
    4. Read the workspace with `readWorkspace(name)`
    5. If NOT `--remove`: show `p.multiselect()` with agent hook plugins (map `agentHookPlugins` to `{ value: plugin.id, label: plugin.label }`)
    6. For each selected plugin, for each repo in workspace where `mode === "worktree"`:
       - If `--remove`: call `plugin.remove(repo.task_path)`
       - Else: call `plugin.install(repo.task_path, workspace.name)`
    7. Show summary: `p.log.success()` with count of repos and plugins processed

    Handle `p.isCancel()` on all prompts — exit gracefully.

    Register in `src/index.ts`:
    - Add `import { installCommand } from "./commands/install"`
    - Add `program.addCommand(installCommand)` before the completion command (which must remain last)
