# v0.17.1 Functional Coverage Readiness

**Status:** Pre-finalization evidence for Phase 88. This document supports release review only; archive, tag, publish, cleanup, and `$gsd-complete-milestone` are not part of Phase 88 execution.

## Terms

- **green suite**: the configured test commands exit successfully.
- **covered source**: the canonical `.coverage/coverage-final.json` report contains nonzero statement or function hits for targeted source files.
- **functional confidence**: the covered source is mapped to the Phase 85-87 behavior areas that v0.17.1 needs for release confidence, with remaining gaps explicitly classified.

## Canonical Command Path

Run the local readiness path in this order:

```bash
bun run coverage
bun run verify:gates
bun run verify
```

`bun run coverage` refreshes the canonical `.coverage/coverage-final.json` artifact. `bun run verify:gates` evaluates inventory, mapped-test, coverage artifact, coverage sentinel, and functional readiness findings. `bun run verify` remains the umbrella command and runs coverage before gates.
